# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Do NOT open a public issue.**

Instead, please email or DM the maintainer directly. We will respond within 48 hours and work with you to understand and address the issue.

## What to Include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response Timeline

- **48 hours** - Initial acknowledgment
- **7 days** - Assessment and fix plan
- **30 days** - Patch release (if applicable)

Thank you for helping keep npi and its users safe.
